# urban_street
Repositório para a atividade do 3° Ciclo avaliativo 
URBAN STREET: Vista o Estilo da Rua
O projeto consiste no desenvolvimento de uma loja virtual de roupas streetwear para a Urban Street, uma marca focada em moda urbana autêntica, com produtos como hoodies, camisetas oversized e acessórios. O objetivo do site é apresentar a marca, destacar sua coleção de produtos, reforçar a atitude e autenticidade do estilo streetwear, e facilitar a visualização dos itens para compra, utilizando HTML, CSS e Bootstrap  para criar uma página moderna, responsiva e alinhada à identidade visual urbana e ousada da marca.
